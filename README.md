# ClubeDoLivro

## Introdução
- Este projeto consiste em um clube do livro com varias funcionalidades dentre elas voce pode emprestar livros, adicionar livros e adicionar amigos.

![](https://i.imgur.com/sLBSPbU.gif)


# Funcionalidades
**Cadastro de Amigos**; O sistema permite cadastrar amigos.
**Edicao de Amigos**: O sistema permite editar amigos.
**Exclusao de Amigos**: O sistema permite excluir amigos.
**Visualizacao de Amigos Cadastrados**: O sistema permite visualizar os amigos cadastrados no sistema.
**Cadastro de Caixas**: O sistema permite cadastrar caixas.
**Visualizacao de Caixas Cadastradas**: O sistema permite visualizar caixas cadastras no sistema.
**Cadastro de Revistas**: O sistema permite cadastrar revistas
**Edicao de Revistas**: O sistema permite editar revistas.
**Exclusao de Revistas**: O sistema permite excluir revistas.
**Visualizacao de Revistas**: O sistema permite visualizar revistas cadastradas.
**Cadastro de Emprestimo**: O sistema permite cadastrar o emprestimo de uma revista para um amigo.
**Visualizacao de Emprestimos**: O sistema permite visualizar os emprestimos cadastrados
**Verificacao de Entrada**: O sistem valida as entradas em cada cadastro realizado.
